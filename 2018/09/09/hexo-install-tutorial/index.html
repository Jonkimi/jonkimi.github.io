Hexo 安装教程

发表于 2018-09-09 | 更新于 2018-11-17 |
本文是一篇简单的Hexo的安装教程，包括安装`hexo-renderer-markdown-it-plus`渲染插件支持注脚与上下标，安装`hexo-math`插件支持`mathjax`显示数学公式，以及安装评论系统DISQUS

依赖软件

  • Git
  • Node.js

Git 安装[1]

  • Windows: 下载并安装 git。
  • Mac: 使用 Homebrew, MacPorts or installer 安装。
  • Linux (Ubuntu, Debian): sudo apt-get install git-core
  • Linux (Fedora, Red Hat, CentOS): sudo yum install git-core

Node.js 安装

  • Windows: 下载最新 LTS 包安装即可，或使用 nvm-window 安装。
  • Linux: 参考官网使用 PackageManager 安装，或使用 Node Version Manager 安装。

插件安装

Hexo 的默认渲染插件是 hexo-renderer-marked，缺少很多功能，比如不支持 GFM、上下标、注脚、emoji 等，使用 hexo-renderer-markdown-it 加强版补足。[2]
